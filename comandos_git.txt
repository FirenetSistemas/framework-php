Verificar versão:
git --version

Cadastrar o nome e o email do autor do projeto:
git config --global user.name "seu nome aqui"
git config --global user.email "seu email aqui"

Iniciar o projeto Git dentro da pasta do projeto a ser iniciado:
git init

Para verificar os novos arquivos que não tá no git:
git status

Para adicionar novos arquivos no git, no caso "." para adicionar todos:
git add .

Para fazer um commit dos arquivos adicionados no git:
git commit -m "arquivos atualizados"
